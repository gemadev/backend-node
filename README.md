# Desafio clase 42: Testeo de funcionalidades

## Pasos
1) Instalar dependencias:
````
npm i
````

2) Iniciar servidor
````
npm start
````

3) Test AXIOS
````
npm run test:axios
````
4) Test MOCHA, CHAI, SUPERTEST
````
npm run test:mocha
````
